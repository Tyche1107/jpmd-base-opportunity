A case study: JPMorgan JPMD on Base

1/ The thesis:

Public blockchains thrive on composability.

DeFi protocols integrate with each other.

Developers build on open primitives.

Permissioned tokens break all of this.

To use JPMD, you need:
1. JPMorgan approval
